# Markdown Blog/News Site

## What This Is

A static website that displays blog/news entries on the homepage, with each entry viewable on its own page. New entries are added by placing markdown files in a content directory. Designed for simplicity, maintainability, and extensibility.

## Core Value

Content-first publishing: Anyone can add a new entry by dropping a markdown file into the content directory—no code changes required.

## Requirements

### Validated

(None yet — ship to validate)

### Active

- [ ] Homepage lists all blog/news entries from markdown files
- [ ] Each entry links to a detail page displaying its content and metadata
- [ ] New entries appear automatically when markdown files are added
- [ ] Content directory uses frontmatter for metadata (title, date, etc.)
- [ ] Basic site styling and responsive layout
- [ ] Tags/categories, RSS, and theming as enhancements

### Out of Scope

- Dynamic comments (handled externally or in future)
- User authentication (not required for content viewing)
- WYSIWYG editing (content is markdown-only for v1)
